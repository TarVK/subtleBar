    Appending data
    The element listens to data being appended by default, so it can change the scrollbar size when elements are added
    When data is added to the original scrollElement, it will be transfered to the scrollContent
    If data is added to the scrollContent, by default, the scrollbar size will not be updated
    In order to enable this, the option detectDescendantAppend:true can be added, this feature is not properly tested however
